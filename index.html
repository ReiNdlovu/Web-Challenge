Summary: Latitude vs. X

The goal of this web design project was to analyze how weather changes as you get closer to the equator. To complete this analysis, we first pulled data from the OpenWeatherMap API to assemble a dataset on over 500 cities.

After gathering the dataset, I used Matplotlib to plot various aspects of the weather vs. latitude. The factors I looked at included: temperature, cloudiness, wind speed, and humidity. This site provides the source data and visualizations created as part of the analysis, as well as explanations and descriptions of any trends and correlations witnessed.
